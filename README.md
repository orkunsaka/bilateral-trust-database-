## Important Notes for New Visitors (Human or AI)

1. **Read `CLAUDE.md` first.** It contains fundamental rules that govern all work in this directory — especially around data safety and file management.
2. **Never delete data or programs.** When reorganizing, always COPY files from `legacy/` into new folders. Do not move or delete originals.
3. **Stay within this folder.** Do not navigate or operate on directories above this project root.
4. **Log all work.** At the end of each session, update `log/progress_log.md` with a summary of what was done.
5. The `legacy/` folder is the canonical source of truth for the original state of all files.
